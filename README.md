## Introduction

SolidOak is a simple IDE for Rust.

## Build Instructions

### Debian-based Linux

```Shell
apt-get install autoconf libgtk-3-dev libglib2.0-dev libcairo2-dev cmake
cargo build
```

### OSX

```Shell
brew install autoconf gtk+3 cmake
cargo build
```
